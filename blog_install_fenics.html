How to Install FEniCS

13/11/2020

FEniCS is an open-source python package that uses the finite element method to solve partial differential equations. This package is very versatile as the user is able to define the variational problem and can also use mesh made through open source software such as gmsh. The FEniCS website does provide installation instructions1 however I found them unclear at times and therefore decided to make this post.
